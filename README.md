Victor's vimrc
=============

Author: Victor Fu <supergothere@gmail.com>

Installation
------------

```
git clone https://github.com/victorfu/vimrc.git ~/.vim
```

```
ln -s ~/.vim/vimrc ~/.vimrc
```

```
vim +PluginInstall +qall
```
